# Gestión de Estudiantes con Menú Interactivo (Python)

Programa sencillo y educativo en **Python puro** que permite gestionar una lista de estudiantes desde la consola mediante un menú interactivo.

### Qué hace este código
- Registrar estudiantes (nombre y programa académico)
- Mostrar todos los estudiantes registrados
- Buscar un estudiante por nombre (insensible a mayúsculas)
- Eliminar un estudiante por nombre
- Guarda todos los datos de forma permanente en el archivo `estudiantes.txt`
- Maneja errores (archivo no encontrado, etc.) con `try-except`
- Usa solo funciones y Python estándar → **sin librerías externas**
